# Webify NL - Landing Page

Landing page profesional y minimalista para **Webify NL**, una agencia de diseño y desarrollo web de alta gama.

🔗 **Sitio Web Publicado:** [https://eugeniocvz222-sketch.github.io/Webify-NL-/](https://eugeniocvz222-sketch.github.io/Webify-NL-/)

---

## 🎨 Características del Proyecto

- **Estética Minimalista y Limpia:** Diseño oscuro navy sobrio enfocado en la legibilidad y la jerarquía visual, sin distracciones de neón ni blobs de color.
- **100% Responsivo:** Adaptabilidad fluida para dispositivos móviles, tablets y pantallas de escritorio.
- **Iconografía Integrada:** Uso unificado de la librería de **Bootstrap Icons** mediante CDN.
- **Animaciones al hacer Scroll:** Efectos suaves de aparición de contenido (`reveal-on-scroll`) utilizando el API nativo `IntersectionObserver`.
- **Formulario de Contacto Funcional:** Configurado con **Formspree** mediante envío asíncrono (AJAX / Fetch API) para mostrar un mensaje de confirmación sin recargar la página.
- **Botón Flotante de WhatsApp:** Acceso directo con tooltip interactivo y mensajes de WhatsApp predefinidos en los botones de cotización.

---

## 🛠️ Tecnologías Utilizadas

- **Estructura:** HTML5 Semántico
- **Estilos:** CSS3 Puro (Variables CSS, Flexbox y Grid)
- **Lógica e Interacciones:** JavaScript Vanilla (JS nativo)
- **Iconografía:** Bootstrap Icons
- **Tipografía:** Google Fonts (Outfit y Plus Jakarta Sans)
